# Voting App

An app/library for using various voting methods. To a certain extent, reinventing [Condorcet.vote](https://www.condorcet.vote/Condorcet_Methods) in React/Typescript. Will support at least FPTP and IRV, and hopefully plugin extensibility for other methods. Must also note the downsides of various methods from [Building A Better Ballot](https://ncase.me/ballot/).

See also https://www.youtube.com/watch?v=yhO6jfHPFQU
